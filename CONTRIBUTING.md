# Contributing

Thanks for helping improve Diamond Shader Lab. Open an issue before substantial visual or architectural changes. Keep rendering code deterministic, preserve `reference/`, and avoid remote runtime assets.

1. Fork and create a focused branch.
2. Run `npm install`, `npm run typecheck`, `npm run lint`, `npm run build`, and `npm run verify:export`.
3. Describe visual changes, performance implications, and tested browsers in the pull request.
